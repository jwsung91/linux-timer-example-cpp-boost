# linux timer example using cpp,boost

timer test program using cpp and boost library

## Test

### How to build 

```shell
mkdir build
cd build
cmake -DCMAKE_BUILD_TYPE=Release ..
make
```

### How to test

``` shell
# Usage: ./linux_timer <delay_seconds> <repeat_count>
# 3 seconds 5 times
./linux_timer 3 5
```
